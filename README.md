# Django Rest-Framework + Vue.JS
### เปิด git bash
```
- git clone https://github.com/joblackpoc/rest-vue.git
```
### เปิด VScode
```
- cd rest-vue
```
### เปิด new terminal
```
- cd rest-vue
- pip install -r requirements.txt
- cd backend
- python manage.py runserve
```
### เปิด new terminal
```
- cd rest-vue
- npm install
- cd frontend
- npm run serve
- เปิด browser localhost:8000/api/student
- เปิด browser localhost:8080
```
